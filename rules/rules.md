# Project-specific rules

- Репозиторий учебный: каждый урок самодостаточен, у него свой venv и `requirements.txt`; не смешивать зависимости между уроками.
- Версии библиотек намеренно старые и зафиксированы под материалы курса — не обновлять без необходимости.
- Файлы `*_blank` — заготовки для выполнения ДЗ, `*_result` — эталонные решения; не путать.
- `.env` файлы закоммичены (вопреки `.gitignore`) — не добавлять туда реальные секреты и не публиковать.
- Вся проза и коммуникации — на русском.
