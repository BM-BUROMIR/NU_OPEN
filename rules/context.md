# Project context

Учебный репозиторий с домашними заданиями и материалами онлайн-курса по интеграции GPT/LLM в приложения (формат «GPT Professional» / «ChatGPT Professional»). Это не форк одного OSS-проекта, а набор независимых уроков: каждая папка `lesson_*` / `lessonN` — отдельный мини-проект (часто в вариантах Lite/Pro и `_home_`). Темы: вызов OpenAI API, retrieval по своей базе (LangChain + FAISS), Telegram-боты, FastAPI-бэкенды, Django-приложения, базовая Android-разработка.

Часть экосистемы UNITE (`~/unite/`).
